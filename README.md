# Battle Pass 數值平衡模擬器 (BP Calculator)

一款專為遊戲營運與數值企劃設計的 Battle Pass (通行證) 平衡工具。透過此工具，你可以快速分配各節點的難度比例，並即時計算對應的押量目標、免費獎勵與付費獎勵價值。

## 🚀 功能亮點

- **多種難度曲線**：支援「線性 (Linear)」、「成長 (Growth)」與「陡峭 (Steep)」三種預設曲線，自動生成建議比例。
- **手動微調模式**：保留高度靈活性，可隨時切換至手動模式自定義每個節點的數值。
- **累積進度計算**：自動根據比例計算單層目標與累積總目標 (如總押量、總積分)。
- **兩線價值分配**：同步計算「免費線」與「付費線」在各個節點的對應價值。
- **內建防呆檢查**：
  - **比例檢查**：自動檢查各節點比例加總是否等於 100%。
  - **價值邏輯檢查**：若偵測到「付費獎勵價值 < 免費獎勵價值」時會發出警告，確保付費吸引力。

## 🛠️ 使用說明

1. **開啟工具**：直接使用瀏覽器開啟 `index.html` 即可運行（無需伺服器環境）。
2. **設定基礎參數**：
   - **總目標押量**：完成整個 Pass 所需的總進度。
   - **免費獎勵佔比**：通常以總押量的一定百分比 (如 1%) 作為免費獎勵的價值總額。
   - **付費獎勵總價值**：該賽季付費線所提供的總價值。
3. **選擇曲線**：
   - 選擇預設曲線並輸入節點總數，系統會自動產出比例。
   - 若要自行填寫，請切換至 **Manual** 並以逗號隔開各節點比例。
4. **生成清單**：點擊「立即計算」即可獲得完整的數值分配表。

## 📊 核心運算邏輯

- **單層目標** = `總目標` × `該層比例`
- **累積進度** = `前一層累積進度` + `當前單層目標`
- **免費獎勵價值** = (`總目標` × `免費獎勵佔比`) × `該層比例`
- **付費獎勵價值** = `付費總價值` × `該層比例`
